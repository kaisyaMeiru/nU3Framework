## Policies Enforced
1.Documentation Access: DevExpress 정식 HTML 문서를 참조합니다.
2.Example Validation: 공식 샘플 프로젝트(Example_CS)의 예제 코드를 참고합니다.
3.Syntax Guidance: 컨트롤별 API와 속성의 올바른 문법 사용을 안내합니다.

## Instructions
1.Do not rely on memory or partial code ? DevExpress 컨트롤의 사용법은 버전에 따라 다를 수 있습니다. 항상 정식 문서를 확인하십시오.
2.Check the Official Documents:
	reference\DevExpress.WindowsForms.v23.2\cd7891.htm 파일이 루트 문서입니다.
	여기서 원하는 컨트롤 이름으로 검색하면 관련 API, 속성, 이벤트 정보를 바로 확인할 수 있습니다.
3.Use Example Projects:
	reference\Example_CS 폴더에는 정식 예제가 포함되어 있으며, 각 컨트롤별 사용법을 코드로 검증할 수 있습니다.

4.Interpret Results:
	문서 또는 예제에서 일치하는 코드가 있다면 해당 API는 사용 가능합니다.
	존재하지 않거나 버전 차이가 있는 경우, 문서의 “Version Notes”를 확인하여 지원 버전을 파악하십시오.
